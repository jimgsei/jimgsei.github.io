Integración para Desarrolladores

Endpoint Oficial

GET https://datos.juntadeandalucia.es/api/v0/rael/search

Ejemplo con JavaScript (Fetch)

const params = new URLSearchParams({
  entity: 'MUNICIPIO',
  province: 'PROVINCIA SEVILLA', 
  typology: '-',
  format: 'json'
});

fetch(`https://datos.juntadeandalucia.es/api/v0/rael/search?${params}`)
  .then(response => response.json())
  .then(data => {
    console.log('Resultados:', data.results);
  })
  .catch(error => console.error('Error:', error));

Ejemplo cURL

curl -X 'GET' \
  'https://datos.juntadeandalucia.es/api/v0/rael/search?entity=MUNICIPIO&province=-&typology=-&format=json' \
  -H 'accept: application/json'

* Los parámetros marcados como opcionales en el formulario deben enviarse como guión - si no se desea filtrar por ellos.
